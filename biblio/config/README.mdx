## **default_inputs**

| Parameter | Purpose                                            | Sensible Values | Effects When Varied                                                                                         |
| --------- | -------------------------------------------------- | --------------- | ----------------------------------------------------------------------------------------------------------- |
| **I_I**   | Baseline drive to inhibitory neurons               | 0.6–1.5         | Low → weak inhibition & synchrony; Mid → stable irregular gamma; High → suppressed E firing, rhythm weakens |
| **I_E**   | Baseline drive to excitatory neurons (unused here) | 0.5–1.5         | Low → weak E drive; High → increased excitability/bursts                                                    |
| **noise** | Additive noise to membrane potential               | 0.3–1.0         | Low → overly synchronized; Mid → realistic irregularity; High → ragged firing, reduced gamma power          |

## **plotting**

| Parameter      | Purpose                | Sensible Values | Effects When Varied                        |
| -------------- | ---------------------- | --------------- | ------------------------------------------ |
| **start_time** | Start of raster window | 300–500 ms      | Earlier windows include transient dynamics |
| **stop_time**  | End of raster window   | > start_time    | Only visualization impact                  |

## **simulation timing**

| Parameter | Purpose                        | Sensible Values | Effects When Varied                                   |
| --------- | ------------------------------ | --------------- | ----------------------------------------------------- |
| **dt**    | Numerical integration timestep | 0.05–0.1 ms     | Larger → instability; Smaller → more accurate, slower |
| **T**     | Total simulation time          | 300–2000 ms     | Short → unstable stats; Long → cleaner statistics     |
| **seed**  | Random number generator seed   | any int         | Affects trajectory, not regime                        |

## **network size**

| Parameter | Purpose                      | Sensible Values | Effects When Varied                                |
| --------- | ---------------------------- | --------------- | -------------------------------------------------- |
| **N_E**   | Number of excitatory neurons | 200–2000        | Small → noisy gamma; Large → smoother oscillations |
| **N_I**   | Number of inhibitory neurons | N_E / 4         | Same effects as above; ratio matters               |

## **synaptic weights**

| Parameter | Purpose                  | Sensible Values | Effects When Varied                                                    |
| --------- | ------------------------ | --------------- | ---------------------------------------------------------------------- |
| **g_ei**  | E→I synaptic strength    | 1.0–2.0         | Low → weak I drive & hypersynchrony; High → overactivated I population |
| **g_ie**  | I→E synaptic strength    | 1.8–3.0         | Low → weak rhythm; Mid → strong gamma; High → E firing suppressed      |
| **g_ee**  | E→E recurrent excitation | 0.1–0.5         | Too high → synchrony and runaway bursts                                |
| **g_ii**  | I→I recurrent inhibition | 0.1–0.5         | High → damped oscillation; Low → tighter I timing                      |

## **connectivity probabilities**

| Parameter | Purpose      | Sensible Values | Effects When Varied                                       |
| --------- | ------------ | --------------- | --------------------------------------------------------- |
| **p_ee**  | E→E sparsity | 0.02–0.15       | Too sparse → weak propagation; Too dense → over-coherence |
| **p_ei**  | E→I sparsity | 0.05–0.2        | Higher → stronger I recruitment                           |
| **p_ie**  | I→E sparsity | 0.02–0.15       | Higher → stronger inhibition → lower firing rates         |
| **p_ii**  | I→I sparsity | 0.02–0.15       | Higher → damped or smeared oscillation                    |

## **synaptic delays**

| Parameter    | Purpose                      | Sensible Values | Effects When Varied                                      |
| ------------ | ---------------------------- | --------------- | -------------------------------------------------------- |
| **delay_ei** | E→I delay                    | 0.3–1.0 ms      | Shorter → faster gamma                                   |
| **delay_ie** | I→E delay (frequency setter) | 1.0–1.5 ms      | Short → higher frequency; Long → slower or unstable PING |
| **delay_ee** | E→E delay                    | 0.3–1.0 ms      | Longer → smeared excitation timing                       |
| **delay_ii** | I→I delay                    | 0.3–1.0 ms      | Longer → desynchronization                               |

## **LIF membrane + intrinsic parameters**

| Parameter        | Purpose                  | Sensible Values | Effects When Varied                               |
| ---------------- | ------------------------ | --------------- | ------------------------------------------------- |
| **V_init**       | Initial membrane voltage | −70 to −60 mV   | Mostly affects transients                         |
| **E_L**          | Leak reversal            | −70 to −60 mV   | Shifts resting voltage                            |
| **E_e**          | Excitatory reversal      | ~0 mV           | Controls EPSP amplitude                           |
| **E_i**          | Inhibitory reversal      | −75 to −85 mV   | Controls IPSP amplitude                           |
| **C_m_E, C_m_I** | Membrane capacitance     | 0.5–2.0         | Higher → slower membrane dynamics                 |
| **g_L_E, g_L_I** | Leak conductance         | 0.03–0.15       | Higher leak → less excitability                   |
| **V_th**         | Spike threshold          | −55 to −48 mV   | Lower → higher firing; Higher → suppressed firing |
| **V_reset**      | Reset potential          | −70 to −60 mV   | Affects recovery dynamics                         |
| **t_ref_E**      | E refractory period      | 2–4 ms          | Longer → slower gamma                             |
| **t_ref_I**      | I refractory period      | 1–2 ms          | Shorter → tighter I spikes                        |

## **synaptic time constants**

| Parameter    | Purpose    | Sensible Values | Effects When Varied                                          |
| ------------ | ---------- | --------------- | ------------------------------------------------------------ |
| **tau_ampa** | AMPA decay | 1.5–3 ms        | Shorter → sharper E volleys → faster gamma                   |
| **tau_gaba** | GABA decay | 5–8 ms          | Short → high-freq gamma; Long → slower beta-ish oscillations |

## **connectivity scaling**

| Parameter                | Purpose                                      | Sensible Values  | Effects When Varied                                                 |
| ------------------------ | -------------------------------------------- | ---------------- | ------------------------------------------------------------------- |
| **connectivity_scaling** | Normalizes synaptic input by population size | `one_over_N_src` | Turning off causes larger networks to become unrealistically strong |

## **heterogeneity**

| Parameter                  | Purpose                 | Sensible Values | Effects When Varied                                                            |
| -------------------------- | ----------------------- | --------------- | ------------------------------------------------------------------------------ |
| **V_th_heterogeneity_sd**  | Threshold variability   | 0–2 mV          | Low → risk of synchrony; Moderate → realistic irregularity; High → poor coding |
| **g_L_heterogeneity_sd**   | Leak variability        | 0–0.2           | Too high → inconsistent firing patterns                                        |
| **C_m_heterogeneity_sd**   | Capacitance variability | 0–0.2           | Too high → poor PING alignment                                                 |
| **t_ref_heterogeneity_sd** | Refractory variability  | 0–0.5 ms        | Moderate breaks synchrony; Too high → desynchronization                        |
