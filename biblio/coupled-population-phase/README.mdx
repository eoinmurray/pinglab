# Coupled Populations Phase Dependence

We investigate whether the relative phase between two coupled oscillating populations determines how
effectively spikes propagate between them. With identical synaptic weights and
input volleys, downstream response varies >4× depending on phase
alignment—maximal when input arrives during low inhibition, minimal during high
inhibition.
