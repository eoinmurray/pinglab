# Phase Dependence

Gamma oscillations create phase-dependent gating of inputs. The phase-gain curve
shows that identical pulses delivered at different gamma phases produce
different responses—inputs during low inhibition (troughs) trigger more spikes
while inputs during high inhibition (peaks) are suppressed.
