## Delivering a Pulse | Raster Plots Across Gamma Phases

Network responses to identical input pulses delivered at different gamma phases.
Each raster demonstrates how the same pulse produces different spiking patterns
depending on when it arrives relative to the ongoing oscillation.

We select 10% of E neurons as targets and deliver brief current pulses (width =
5 ms, amplitude = 0.5) at 100 different times spanning 500–700 ms (one gamma
cycle period). For visualization, we save raster plots for every 10th pulse.
Each raster shows the 500–1000 ms window with the pulse timing indicated.

Pulses arriving during inhibitory troughs (low $g_i$) should trigger robust
responses with many target neurons firing. Pulses arriving during inhibitory
peaks (high $g_i$) should be largely suppressed. The spatial pattern of
responsive neurons should shift with pulse timing.

## Delivering a Pulse | Phase-Dependent Gain Curve

This curve demonstrates that gamma oscillations implement a phase-dependent gain
control mechanism.

For each of 100 pulse times spanning 500–700 ms, we compute Δ-spikes = (spikes
in post-window) − (spikes in pre-window) for the target E neurons. The
pre-window is 20 ms before the pulse; the post-window is 100 ms after. We
overlay the normalized inhibitory conductance (LFP proxy) from the baseline
simulation to show the phase relationship.

Δ-spikes should be anti-correlated with inhibitory conductance: maximum response
when $g_i$ is at its trough, minimum response when $g_i$ is at its peak.
