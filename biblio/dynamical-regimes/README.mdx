# Dynamical Regimes

## TL;DR Outcome
{/* one paragraph */}

The network transitions from asynchronous irregular (AI) dynamics to oscillating
(OSC) gamma rhythms as either the excitatory-to-inhibitory conductance ($g_{ei}$)
or the external drive ($I_E$) increases. Raster plots illustrate this shift,
with early plots showing irregular firing patterns and later plots revealing
organized rhythmic activity. Both parameters control the transition by different
mechanisms: $g_{ei}$ strengthens the E→I coupling that drives PING oscillations,
while $I_E$ increases overall network excitability.

## Experimental Design
{/* the design */}

Two parameter sweeps are performed to observe their effects on network dynamics.

### Network Configuration

- Network size: 800 E cells, 200 I cells (4:1 ratio)
- Simulation duration: 1000 ms (discard first 500 ms for raster)
- External input: Tonic + noise ($I_I = 0.9$, noise_std = 0.6)
- Base synaptic conductances: $g_{ie} = 2.5$, $g_{ee} = 0.3$, $g_{ii} = 0.3$

### Sweep 1: $g_{ei}$ (E→I conductance)

- **Swept parameter**: $g_{ei}$
- **Range**: 1.2 to 1.7 nS (10 values, linear spacing)
- **Fixed**: $I_E = 1.2$

### Sweep 2: $I_E$ (External drive to E neurons)

- **Swept parameter**: $I_E$
- **Range**: 1.15 to 1.25 (10 values, linear spacing)
- **Fixed**: $g_{ei} = 1.4$

### Output

- Raster plots for visual inspection of regime transitions
- Time window: 500–1000 ms (post-transient)

## Discussion

### Background

### Asynchronous-Irregular (AI) Regime
Neurons fire irregularly and independently with no temporal structure. This occurs when inhibitory feedback is weak.

**Signatures**:
- Flat power spectrum (no dominant frequency)
- High interspike interval variability (CV $> 0.7$)
- Low population synchrony
- Activity evenly distributed across time

### Oscillating (OSC) Regime
Strong inhibitory feedback creates rhythmic gamma oscillations through an "escape-then-suppress" cycle.

**Signatures**:
- Sharp spectral peak at gamma frequency (30-90 Hz)
- Periodic population activity
- High spike-time synchrony
- Activity concentrated in brief temporal windows

**Transition control**: The regime can be controlled by either **excitatory-to-inhibitory conductance** $g_{ei}$ or **external drive** $I_E$. Increasing $g_{ei}$ strengthens the PING mechanism, while increasing $I_E$ raises overall excitability—both push the network from AI toward OSC.
