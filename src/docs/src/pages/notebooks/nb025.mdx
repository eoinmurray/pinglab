## Introduction

[023 — PING fundamentals](/notebooks/nb023/) and [024 — Inhibition-gated sparsity](/notebooks/nb024/) characterise coba and ping with only the two synaptic time constants the COBA model has by default: AMPA ($\tau = 2$ ms) and GABA ($\tau = 9$ ms). The longest physical time constant in the network is ~9 ms — an order of magnitude shorter than the cognitively-relevant working-memory window. Anything that requires the network to remember an input across more than a few tens of milliseconds is structurally impossible to encode in the current architecture's dynamics.

This entry adds a third synaptic channel — an NMDA-like slow excitatory current — as a new option in the oscilloscope: *--slow-syn*, optional *--tau-nmda* (default 100 ms) and *--slow-syn-gain* (default 0.5). The slow channel runs in parallel with the existing AMPA channel on every excitatory projection onto E cells, reuses the same $W$ matrices (no new trainable parameters), and adds a single per-layer slow-conductance buffer. Gain 0.0 is bit-identical to plain COBA; non-zero gain layers a long-decay residue on top.

This first cell is purely observational — take the nb024-trained coba and ping baselines, freeze every weight, and replay each on one MNIST trial twice: once with slow-syn off (parity check) and once at gain 0.5. Same weights, same input, same trial. The question is whether the slow channel meaningfully changes the spike statistics, given that the trained $W$ matrices were never optimised to use it.

## Method

The slow channel updates in parallel with the fast AMPA channel:

$$
g^{e,\text{slow}}_{t+1} = e^{-\Delta t/\tau_\text{NMDA}}\left(g^{e,\text{slow}}_t + \alpha \cdot W s_t\right)
$$

where $W s_t$ is whichever excitatory drive feeds layer $\ell$ on this step (input-to-E for layer 1, recurrent E→E and inter-layer feedforward for deeper layers), $\alpha$ is *--slow-syn-gain*, and $\tau_\text{NMDA}$ is *--tau-nmda*. The membrane equation gains one term:

$$
C_m \frac{dV}{dt} = -g_L(V-E_L) - (g^e + g^{e,\text{slow}})(V-E_e) - g^i(V-E_i)
$$

with the same reversal potential $E_e = 0$ mV for both fast and slow excitatory channels.

**No new trainable parameters.** The slow channel reuses the existing $W_\text{in}$, $W_\text{ee}$, and $W_\text{ff}$ matrices. One extra conductance tensor per E layer, one extra add and one extra scalar multiply per timestep — about 5% per-step overhead.

**This cell:** replay the trained coba and ping baselines (from nb024) on MNIST digit 0 sample 0 for 400 ms. Two slow-syn gains: 0.0 (parity with nb024) and 0.5 (Wang/Compte-style NMDA contribution). $\tau_\text{NMDA} = 100$ ms.
