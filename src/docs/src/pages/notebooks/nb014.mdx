## Discussion

*To be written once results land.*

A side experiment swapped CE for L2 (MSE between mean output membrane and one-hot targets) at the same recipe. Cuba held (84% vs 87% under CE) but coba and ping collapsed: coba's hidden layer never bootstrapped firing (rate_e ≈ 0.3 Hz, accuracy 48%), and ping diverged with output magnitudes that overshot the one-hot target by an order of magnitude. Per-model retunes of *--readout-w-out-scale* and *--lr* didn't recover. The likely reason is structural: CE's softmax provides per-step "lower me / raise me" gradient to *every* output class; one-hot MSE only supplies non-zero gradient on the correct class when other outputs sit at zero, which the biophysical models do at init. Reverting to CE here.
