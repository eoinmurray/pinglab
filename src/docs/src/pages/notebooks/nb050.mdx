## Abstract

One network ([COBANet](/articles/ar003/)) can run as the gamma rhythm **PING** or as the [Vreeswijk-Sompolinsky](https://doi.org/10.1126/science.274.5293.1724) **asynchronous-irregular** state. With everything else harmonised — connectivity, weights, $W^{II}$ — the two differ in a single control: the **correlation of the input** across cells. Shared (correlated) drive gives gamma; per-cell independent drive gives the balanced state.

## Methods

### Model parameters

The two models are identical COBANet except for the input. The table is the full configuration; the only rows that differ — all of them describing the input — are in *italics*.

| Parameter | PING | V&S AI |
|---|---|---|
| Model | COBANet (conductance LIF) | COBANet (conductance LIF) |
| $N_E$ / $N_I$ | 1024 / 256 | 1024 / 256 |
| $\Delta t$ / $T$ | 0.25 ms / 1000 ms | 0.25 ms / 1000 ms |
| $\tau_{E}$ / $\tau_{I}$ (AMPA / GABA) | 2 / 9 ms | 2 / 9 ms |
| Recurrent connectivity | sparse fixed-fan-in, $p = 0.2$ | sparse fixed-fan-in, $p = 0.2$ |
| $W^{EI}$ (E→I) | $\mathcal{N}(0.6, 0.18)$ μS | $\mathcal{N}(0.6, 0.18)$ μS |
| $W^{IE}$ (I→E) | $\mathcal{N}(3.0, 0.9)$ μS | $\mathcal{N}(3.0, 0.9)$ μS |
| $W^{II}$ (I→I) | none | none |
| External drive | *shared 1024-neuron layer via $W_\text{in}$* | *per-cell independent Poisson, bypassing $W_\text{in}$* |
| Drive rate | *20 Hz* | *45 Hz (E), 8 Hz (I)* |
| Input correlation across cells | *correlated* | *independent* |

Both models use sparse fixed-fan-in recurrent connectivity at $p = 1 - s = 0.2$ on the E↔I matrices ($W^{EE} = 0$, $W^{II}$ off). Because $p$ is also the fraction of recurrent input any two cells share ($\langle n_\text{shared}\rangle / K = K/N = p$), it sets how strongly the loop can cohere — connectivity is the *internal* counterpart of the input knob. We fix it at a cortically realistic $p = 0.2$ (local E↔I connectivity runs ≈ 0.1–0.5) — dense enough for the recurrent loop to cohere, so connectivity is shared and does not distinguish the regimes.

### Model differences

The models share everything above — connectivity, the E↔I weights, $W^{II}$. The single difference is the **input**: a shared (correlated) drive for PING, per-cell independent (uncorrelated) drive for the V&S cell.

Both are Poisson spike trains converted to a decaying AMPA conductance — the same *kind* of input. They differ only in the **correlation of that drive across cells**. The external excitatory conductance onto E cell $j$ obeys

$$
\tau_E \,\dot g^{\text{ext}}_j(t) = -\,g^{\text{ext}}_j(t) \;+\;
\begin{cases}
\displaystyle\sum_{k=1}^{N_\text{in}} W^{\text{in}}_{jk}\, s_k(t) & \text{(PING)} \\[1.4em]
g_0\, \xi_j(t) & \text{(V\&S AI)}
\end{cases}
$$

where:

- $g^{\text{ext}}_j(t)$ — external excitatory conductance onto E cell $j$;
- $\tau_E$ — AMPA decay time constant (2 ms), common to both;
- $s_k(t) = \sum_m \delta(t - t_k^m)$ — spike train of **shared** input neuron $k$ (Poisson, 20 Hz), read by every E cell;
- $W^{\text{in}}_{jk}$ — dense input weight from input neuron $k$ to E cell $j$; $N_\text{in} = 1024$;
- $\xi_j(t) = \sum_m \delta(t - t_j^m)$ — a **private** Poisson train (45 Hz) unique to cell $j$;
- $g_0$ — conductance increment per private spike (0.38 μS).

The consequence is the cross-cell correlation of the drive:

$$
\operatorname{Corr}\!\big(g^{\text{ext}}_j,\, g^{\text{ext}}_{j'}\big)
\;>\; 0 \ \ \text{(PING, shared layer)},
\qquad
\operatorname{Corr}\!\big(g^{\text{ext}}_j,\, g^{\text{ext}}_{j'}\big)
\;=\; 0 \ \ \text{(V\&S AI, private streams)},
\qquad j \neq j'.
$$

The shared layer is a *common reference* the loop can lock onto; the private streams kick each cell independently, with no common phase. (The drive *rate*, 20 vs 45 Hz, is an incidental magnitude — it sets the operating rates, not the qualitative state.) What both share: neither input carries any rhythm. Both are featureless, steady Poisson (Figure 1), so PING's gamma is generated by the network, not inherited from the drive.

## Next steps

The two regimes are now identical except for the input, and that difference is its **cross-cell correlation** — realised here as a shared input layer (PING) versus per-cell independent streams (V&S). Turning that into a single, continuous control is the open work:

1. **A faithful correlation knob.** The tidy idea — mix a shared common stream with private streams and dial the shared fraction $c \in [0, 1]$ — breaks at the $c = 1$ extreme if "shared" is *one* stream broadcast verbatim: every cell then receives identical input and locks into trivial synchrony (cross-correlation ≈ 0.99, a broad ≈ 74 Hz peak) rather than loop-generated gamma. Real correlated drive is a shared *population*, not one identical signal. The fix is a shared *layer* of $K_\text{shared}$ common input neurons (each cell reading all of them), interpolated against private streams at matched rate — correlated without being identical, so genuine gamma survives. With that knob, the gamma↔asynchronous boundary can be mapped continuously over $c$.
2. **Matched-rate input.** The shared layer runs at 20 Hz and the independent streams at 45 / 8 Hz; equalising the rates would make cross-cell correlation the sole input difference even before the continuous knob is built.
