## Abstract

The same conductance-based network ([COBANet](/articles/ar003/), $N_E = 1024$, $N_I = 256$) produces two opposite regimes depending on how it is wired and driven: the gamma rhythm **PING**, and the [Vreeswijk-Sompolinsky](https://doi.org/10.1126/science.274.5293.1724) **asynchronous-irregular (AI)** state — the balanced regime that explains why cortex fires irregularly (CV ≈ 1) without tuning. **The goal is to find the minimum set of control parameters that lets the one simulator exhibit both states** — to pin down exactly which knobs must change to cross between gamma and the balanced regime, and which are incidental. This entry sets the two side by side: their inputs, their parameters, and their rasters.

## Inputs

Both regimes are driven by Poisson spike trains converted to a decaying AMPA conductance — the same *kind* of input. They differ in one thing: the **correlation of that drive across cells**. The external excitatory conductance onto E cell $j$ obeys

$$
\tau_E \,\dot g^{\text{ext}}_j(t) = -\,g^{\text{ext}}_j(t) \;+\;
\begin{cases}
\displaystyle\sum_{k=1}^{N_\text{in}} W^{\text{in}}_{jk}\, s_k(t) & \text{(PING)} \\[1.4em]
g_0\, \xi_j(t) & \text{(V\&S AI)}
\end{cases}
$$

where:

- $g^{\text{ext}}_j(t)$ — external excitatory conductance onto E cell $j$;
- $\tau_E$ — AMPA decay time constant (2 ms), common to both;
- $s_k(t) = \sum_m \delta(t - t_k^m)$ — spike train of **shared** input neuron $k$ (Poisson, 20 Hz), read by every E cell;
- $W^{\text{in}}_{jk}$ — dense input weight from input neuron $k$ to E cell $j$; $N_\text{in} = 1024$;
- $\xi_j(t) = \sum_m \delta(t - t_j^m)$ — a **private** Poisson train (45 Hz) unique to cell $j$;
- $g_0$ — conductance increment per private spike (0.38 μS).

The consequence is the cross-cell correlation of the drive:

$$
\operatorname{Corr}\!\big(g^{\text{ext}}_j,\, g^{\text{ext}}_{j'}\big)
\;>\; 0 \ \ \text{(PING, shared layer)},
\qquad
\operatorname{Corr}\!\big(g^{\text{ext}}_j,\, g^{\text{ext}}_{j'}\big)
\;=\; 0 \ \ \text{(V\&S AI, private streams)},
\qquad j \neq j'.
$$

This difference is not cosmetic: PING's recurrent loop needs a *common* drive to phase-lock into a rhythm, while the balanced state needs *independent* drive to stay decorrelated. (Driving PING with the AI cell's per-cell independent streams collapses its gamma — the cross-correlation drops from ≈ 0.05 to ≈ 0.01 — so the two regimes necessarily differ in input as well as wiring.) What they share: neither input carries any rhythm. Both are featureless, steady Poisson (Figure 1), so PING's gamma is generated by the network, not inherited from the drive.

## Model parameters

Everything else is identical COBANet; the table is the full difference between the two runs.

Rows where the two regimes differ are in *italics*.

| Parameter | PING | V&S AI |
|---|---|---|
| Model | COBANet (conductance LIF) | COBANet (conductance LIF) |
| $N_E$ / $N_I$ | 1024 / 256 | 1024 / 256 |
| $\Delta t$ / $T$ | 0.25 ms / 1000 ms | 0.25 ms / 1000 ms |
| $\tau_{E}$ / $\tau_{I}$ (AMPA / GABA) | 2 / 9 ms | 2 / 9 ms |
| Recurrent connectivity | *dense* | *sparse, fixed fan-in $K \approx 10$* |
| $W^{EI}$ (E→I) | *$\mathcal{N}(1.5, 0.15)$ μS* | *$\mathcal{N}(0.6, 0.18)$ μS* |
| $W^{IE}$ (I→E) | *$\mathcal{N}(3.0, 0.3)$ μS* | *$\mathcal{N}(3.0, 0.9)$ μS* |
| $W^{II}$ (I→I) | *none* | *$\mathcal{N}(0.4, 0.12)$ μS* |
| External drive | *shared 1024-neuron layer via $W_\text{in}$* | *per-cell independent Poisson, bypassing $W_\text{in}$* |
| Drive rate | *20 Hz* | *45 Hz (E), 8 Hz (I)* |
| Input correlation across cells | *correlated* | *independent* |

The decisive rows are the bottom four: a dense loop with a shared drive and no $I\!\to\!I$ gives gamma; sparse fixed-$K$ wiring with $I\!\to\!I$ and independent drive gives the balanced state.

The contrast is total: $\langle r_E\rangle, \langle r_I\rangle$ = 3.6, 31 Hz (PING) vs 15, 23 Hz (AI); a 31 Hz gamma peak vs none; median ISI CV 0.61 vs 1.02; pairwise correlation 0.047 vs 0.009. The balanced state turns the same hardware that makes gamma into a decorrelated, irregular, Poisson-like regime — the cortical "ground state" against which the rest of this collection's gamma work is measured.
