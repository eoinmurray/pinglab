## Introduction

Single-model companion to [nb003](/notebooks/nb003/). Trains only the [*coba*](/models/#coba) model — conductance-based synapses, dispatched through the PINGNet class with *ei_strength* = 0 so the E→I→E loop is disabled — at *dt* = {cfg.dt ?? "?"} ms on MNIST and publishes its own curves, firing-rate trace, and training video. The second rung on the CUBA → PING ladder. No Δt sweep.

## Method

*coba* (PINGNet with *ei_strength* = 0) with 1024 hidden units, sparse $W_\text{in}$ (density 0.05, scale 0.3), *v_grad_dampen* = 1000, learning rate $\eta$ = {cell.lr ?? "?"}. Trained at *dt* = {cfg.dt ?? "?"} ms for *T* = {cfg.t_ms ?? "?"} ms per trial, on {cfg.max_samples ?? "?"} MNIST samples × {cfg.epochs ?? "?"} epochs. All hyperparameters are literals in *src/pinglab/notebooks/nb008.py*.
