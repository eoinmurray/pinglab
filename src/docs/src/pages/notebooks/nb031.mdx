## Introduction

[030 — Coba + NMDA cannot sustain post-stimulus activity](/notebooks/nb030/) makes a structural claim: with the slow channel on but no inhibitory loop, a recurrently-coupled E population has only two stable outcomes — decay or refractory saturation. The mean-field argument identifies the missing piece — a second (negative-feedback) equation — but doesn't prove that turning that piece on actually fills in the sustained band.

This entry adds the proof. Same 2D sweep, same slope-aware classifier, only one thing changes: *ei_strength*. Run coba (0) and ping (1) side by side. Ping should have a non-empty sustained band that coba doesn't.

## Method

Same setup as [nb030](/notebooks/nb030/) — untrained COBANet, $N_E = ${cfg.n_e}, slow channel on at gain {cfg.slow_syn_gain}. **Input is the MNIST digit-0 (sample 0), pixel-weighted Poisson-encoded** over $N_{IN} = 784$ channels during $[0, ${cfg.t_stim_ms}]$ ms, then silence to ${cfg.t_total_ms} ms — same encoder pipeline as the trained MNIST notebooks ([nb024](/notebooks/nb024/), [nb025](/notebooks/nb025/), [nb027](/notebooks/nb027/)) so the input statistics match what a real trained run sees. The sweep axis $R$ is the *peak* Poisson rate at pixel intensity = 1; mean per-channel rate ≈ $0.18\,R$ for digit 0.

Two models:

| | *ei_strength* | $W_{in}$ mean | Notes |
|---|---|---|---|
| coba | 0 | 0.3 | No I→E feedback; only E cells effectively recurrent |
| ping | 1 | 1.2 | Full E↔I↔E loop with W_ei, W_ie at PING defaults |

Each model uses its own canonical $W_{in}$ (coba's is gentler to compensate for the missing inhibition that would otherwise hold a strong $W_{in}$ in check). Everything else — slow channel, classifier, grid axes — is identical to nb030 so the comparison is one-variable.

**Classifier** (verbatim from nb030):

- **seizure** — $r_2 \geq 120$ Hz.
- **decay** — $r_2 < 1$ Hz OR $r_2 / r_1 < 0.5$ (still actively decaying).
- **sustained** — anywhere else: stable late-window rate in $[1, 120)$ Hz.

where $r_1$, $r_2$ are mean E rate over $[200, 300]$ and $[500, 600]$ ms respectively.

## Discussion

Three observations that go beyond just "ping works":

**1. The sustained band has both lower and upper $W_{ee}$ boundaries.** Below it (W_ee = 0, 0.05, 0.1): the inhibitory loop dominates and the network can't keep itself going post-stim — it decays just like coba. Above it (W_ee = 1.0): inhibition isn't fast enough to prevent runaway, so saturation wins. Inhibition isn't a magic brake; it has its own gain, and the attractor lives in the slice where E and I gains balance.

**2. The sustained band has an upper *input rate* boundary too.** At $W_{ee} = 0.5$ ping sustains at 5–50 Hz input but seizes at 100+ Hz. Strong input charges the slow channel beyond what inhibition can clamp; the network is then pushed past the upper edge of the basin around its sustained attractor and falls into the seizure attractor. This is consistent with how working-memory layers in cortex behave: they integrate evidence up to a saturation point and then "decide" by locking, losing their ability to hold further information.

**3. Ping's sustained activity is structured, coba's saturated activity isn't.** The bottom-left raster in Figure 2 shows the canonical PING gamma rhythm — synchronous bursts every ~10 ms, all 80 plotted cells firing in lockstep but with visible quiet windows between bursts. That's a population that's *integrating* via its inhibitory cycle. Coba's saturated state (Figure 2 in [nb030](/notebooks/nb030/)) is structureless — every cell firing every refractory period, no temporal correlation, no information.

**Together with nb030 these two notebooks settle the question structurally**: persistent-but-bounded post-stimulus activity in a pinglab COBANet *requires* the inhibitory loop. The slow channel is necessary but not sufficient. Inhibition is the structural piece, and within the inhibitory model there's a clearly delineated sustained band over $(W_{ee}, R)$ where the network behaves as a working-memory layer.

## Next steps

- **Quantify the sustained band shape across seeds.** The current grid uses a single seed per cell. The boundary edges may be soft — a higher-tier run with $n = 5$ seeds would let us put error bars on which cells reliably sustain.
- **What about $W_{in}$?** This entry uses each model's native $W_{in}$ as a control. The honest follow-up is: when $W_{in}$ is matched between models, does the ping sustained band shrink or move? That would isolate the contribution of *just* the inhibitory loop vs the input strength.
- **Multi-sample storage.** Can ping's sustained attractor hold two distinguishable bumps? Drive the network with two non-overlapping subsets of input channels — different "patterns" — and check whether the post-stim state retains which pattern was loaded.
