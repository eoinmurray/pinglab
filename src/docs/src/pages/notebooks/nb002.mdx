**PING** — Pyramidal-Interneuron Network Gamma — emerges when excitatory (**E**) and inhibitory (**I**) populations drive each other: E fires, recruits I, I silences E, I decays, E fires again. The cycle repeats at $\sim$30–80 Hz. It only runs when E gets enough external drive; below threshold the network sits in background firing.

### Run parameters

| Parameter | Value | Meaning |
|---|---|---|
| Model | {cfg.model} | E + I populations, conductance-based synapses |
| N_E / N_I | {cfg.n_e} / {cfg.n_i} | excitatory / inhibitory neuron count |
| dt / T | {cfg.dt_ms} / {cfg.sim_ms} ms | step / total time per frame |
| Stim window | {cfg.step_on_ms}–{cfg.step_off_ms} ms | input rate × overdrive inside this interval |
| Input | {input.dataset} d{input.digit} s{input.sample} | Poisson-encoded pixels, {input.base_rate_hz} Hz max |
| Overdrive scan | {scan.min}×–{scan.max}× | *scan_overdrive.mp4* |
| dt scan | {dtScan.min}–{dtScan.max} ms | *scan_dt.mp4* at overdrive {dtScan.fixed_overdrive}× |
| E→I strength scan | {eiScan.min}–{eiScan.max} | *scan_ei.mp4* at overdrive {eiScan.fixed_overdrive}× |
| Frames / FPS | {scan.frames} / {scan.fps} | per video |
| Seed | {cfg.seed} | RNG seed |
| Tier | **{cfg.tier}** | see [Style guide § 9 Run sizing tiers](/styleguide/#9-run-sizing-tiers) |
| Run ID | <span class="font-mono">{nbRunId}</span> | |
