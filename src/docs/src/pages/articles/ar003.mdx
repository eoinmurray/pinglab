## 1. The COBA model

| | equation |
|---|---|
| **E membrane** | $C_m^E\,\dfrac{dV^E}{dt} = -g_L^E (V^E - E_L) - g_e^E (V^E - E_e) - g_i^E (V^E - E_i)$ |
| **I membrane** | $C_m^I\,\dfrac{dV^I}{dt} = -g_L^I (V^I - E_L) - g_e^I (V^I - E_e)$ |
| **Spike + reset** | $s_{t+1} = \mathbb{1}[V \geq V_{\text{th}}], \quad V \leftarrow V_{\text{reset}} \ \text{if } s_{t+1} = 1 \text{ or refractory}$ |
| **E excitatory conductance** $g^E_e$ | $g^E_{e,t+1} = e^{-\Delta t / \tau_{\text{AMPA}}}\, g^E_{e,t} + W_{\text{in}}\, s^{\text{inp}}_t$ |
| **E inhibitory conductance** $g^E_i$ | $g^E_{i,t+1} = e^{-\Delta t / \tau_{\text{GABA}}}\, g^E_{i,t} + W_{\text{ie}}\, s^i_t$ |
| **I excitatory conductance** $g^I_e$ | $g^I_{e,t+1} = e^{-\Delta t / \tau_{\text{AMPA}}}\, g^I_{e,t} + W_{\text{ei}}\, s^e_t$ |
