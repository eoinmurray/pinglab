## Overview

This experiment tests whether **oscillatory phase alignment** and **spike-time
precision** modulate effective communication into a receiver network (the
communication-through-coherence idea), without building a full sender network.

Instead of a sender, we use a **controlled spike-packet generator** that can:

* place packets at specific **phases** of the receiver oscillation
* add controllable **timing jitter** (precision)

We then measure how strongly the receiver responds as a function of **delay
(phase)** and **jitter**.

## Implementation

### Receiver network

* A fixed PING regime (no coupling sweep during the main test).
* We record population-mean inhibitory conductance as the phase proxy.

### Input interface

* Create an input fiber population.
* Each fiber targets a random subset of E neurons (fixed projection).
* Inputs are **spikes only** (no analog sender network).

### Packet generator

For each packet time:

* Each fiber emits a Poisson number of spikes (mean `K`).
* Spike times are uniform in `[t0, t0 + width]`.
* Gaussian jitter is added to each spike time.

### Phase control (delay sweep)

We keep packet times fixed in absolute time and apply a **delay sweep** to the
packets. This sweeps the arrival phase relative to the receiver oscillation
without changing packet generation logic.

### Readout metric

For each packet arrival time, we measure a baseline-subtracted evoked response
in the **targeted E neurons**:

```
R_evoked = spikes_target(post window) - spikes_target(baseline run, same window)
```

We then average across packet times and trials.

## How to run

```
uv run python experiments/coupled-networks/experiment.py
```

Artifacts are written to `content/_artifacts/coupled-networks/`.

## Config snapshot

Key experiment settings (from `experiments/coupled-networks/config.yaml`):

* Packet times: 500–900 ms (10 packets)
* Delay sweep: 0–25 ms (13 steps)
* Jitter levels: 0, 1, 2, 5 ms
* Packet width: 3 ms
* Fibers: 200, targets per fiber: 80
* Readout window: 20 ms (baseline subtraction from baseline run)

## Interpretation

The response curves should show:

* **Phase dependence:** a clear modulation of evoked response vs delay.
* **Precision dependence:** higher jitter reduces overall response and flattens
  the phase modulation.

This is the minimal Fries-style signature: precise packet timing is required to
exploit the oscillatory excitability window.
