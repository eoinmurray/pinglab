## Overview

This experiment establishes the **baseline dynamical regimes** of a PING
(Pyramidal–Interneuron Network Gamma) circuit under systematic parameter
variation.

The goal is not to demonstrate gamma per se, but to **identify, delimit, and
instrument** the transition between asynchronous and oscillatory regimes under
controlled conditions. This provides the reference point for all subsequent
experiments on perturbation, communication, robustness, and control.

Throughout, we emphasize *rate-matched* and *diagnostically grounded*
comparisons, consistent with the framing of PING as a circuit-level control
mechanism rather than a representational theory.

The narrative is a sweep: start with the knobs that define regime, observe the
transition in raw spiking, then confirm it with spectral and timing diagnostics.

## Question and Parameters

We vary three core parameters that jointly determine network state:

- $g_{ei}$ — excitatory → inhibitory synaptic strength  
- $I_E$ — external drive to excitatory neurons  
- $\sigma$ — input noise standard deviation  

Across these dimensions, the network exhibits a transition between:

- **Asynchronous (A)** activity: irregular spiking, weak temporal structure  
- **Oscillatory (OSC)** activity: coherent population rhythms consistent with
  PING dynamics  

The purpose of the sweeps is to map where these regimes occur and how they are
expressed across standard metrics.

## Experiments and Diagnostics

We run the experiments in a fixed order so each step adds evidence without
changing multiple variables at once.

### 1. $g_{ei}$ sweep at fixed input

We sweep $g_{ei}$ while holding $I_E$ fixed to isolate the effect of
E→I feedback strength.

**Readout:** raster plots.
**Interpretation:** Increasing inhibitory feedback stabilizes recurrent.
excitation and induces rhythmic population structure.

### 5. E–I timing and PING verification

We compute cross-correlograms between excitatory and inhibitory populations in
the oscillatory regime.

**Readout:** E–I cross-correlation  
**Interpretation:** Inhibitory spikes reliably lag excitatory spikes by a few
milliseconds, consistent with the PING mechanism.

**How the plot is produced (math):** Let $\{t^E_m\}$ and $\{t^I_n\}$ be excitatory
and inhibitory spike times. For a lag $\tau$, we count E–I pairs whose time
difference falls in a bin around $\tau$:

$$
H(\tau) = \sum_{m,n} \mathbf{1}\!\left[t^I_n - t^E_m \in \left(\tau-\tfrac{\Delta}{2},\,\tau+\tfrac{\Delta}{2}\right)\right],
$$

where $\Delta$ is the bin width. The plotted curve is the normalized
cross-correlogram $H(\tau)/\max_\tau H(\tau)$, with $\tau>0$ indicating I spikes
lagging E spikes.

asynchronous to oscillatory activity, reflecting increased temporal regularity.
