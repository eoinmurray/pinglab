We aim to identify a suitable neuron type for PING models. It should:

- Be easily excitable with intrinsic excitation properties.
- Have a large range of parameters which keep the network in the PING regime.
- Be as easily to understand and simulate as possible.
- Not add to much time complexity to large scale simulations.
- We want input current to trigger neuron spikes, not drive them.

Candidate Neuron Types:

- Leaky Integrate-and-Fire (LIF)
- Hodgkin-Huxley (HH)
- Adaptive Exponential Integrate-and-Fire (AdEx) Brette and Gerstner
- Connor-Stevens Neuron
- FitzHugh Neuron
- MQIF - Multi-Quadratic Integrate-and-Fire - Burgers
- Izhikevich Neuron

For each neuron we will generate the following artifacts:

- Gif of Voltage vs time for various tonic input current levels
- IF curve
- Rhythmicity vs Tonic Input Current plot
- Rasters vs Tonic Input Current plot

# Leaky Integrate-and-Fire (LIF)

The LIF dynamics used in pinglab are:

$$ C_m \frac{dV}{dt} = g_L (E_L - V) + g_e (E_e - V) + g_i (E_i - V) + I_{ext} $$

# Hodgkin-Huxley (HH)

The Hodgkin-Huxley dynamics used in pinglab are:

$$ C_m \frac{dV}{dt} = I_{ext} + g_e (E_e - V) + g_i (E_i - V) - I_{Na} - I_K - I_L $$

$$ I_{Na} = g_{Na} m^3 h (V - E_{Na}), \quad I_K = g_K n^4 (V - E_K), \quad I_L = g_L (V - E_L) $$

$$
\frac{dm}{dt} = \alpha_m(V)(1-m) - \beta_m(V)m, \quad
\frac{dh}{dt} = \alpha_h(V)(1-h) - \beta_h(V)h, \quad
\frac{dn}{dt} = \alpha_n(V)(1-n) - \beta_n(V)n
$$

$$ \alpha_m(V) = 0.1 \frac{V + 40}{1 - e^{-(V + 40)/10}}, \quad \beta_m(V) = 4 e^{-(V + 65)/18} $$

$$ \alpha_h(V) = 0.07 e^{-(V + 65)/20}, \quad \beta_h(V) = \frac{1}{1 + e^{-(V + 35)/10}} $$

$$ \alpha_n(V) = 0.01 \frac{V + 55}{1 - e^{-(V + 55)/10}}, \quad \beta_n(V) = 0.125 e^{-(V + 65)/80} $$

# Adaptive Exponential Integrate-and-Fire (AdEx) Brette and Gerstner

The AdEx dynamics used in pinglab are:

$$ C_m \frac{dV}{dt} = g_L (E_L - V) + g_L \Delta_T e^{(V - V_T)/\Delta_T} - w + I_{ext} + g_e (E_e - V) + g_i (E_i - V) $$

# Connor-Stevens Neuron

The Connor-Stevens dynamics used in pinglab are:

$$ C_m \frac{dV}{dt} = I_{ext} + g_e (E_e - V) + g_i (E_i - V) - I_{Na} - I_K - I_A - I_L $$

# FitzHugh Neuron

The FitzHugh dynamics used in pinglab are:

$$ \frac{dV}{dt} = V - \frac{V^3}{3} - W + I_{ext} + g_e (E_e - V) + g_i (E_i - V) $$

# MQIF - Multi-Quadratic Integrate-and-Fire - Burgers

The MQIF dynamics used in pinglab are:

$$ C_m \frac{dV}{dt} = g_L (E_L - V) + \sum_{i} a_i (V - V_{r,i})^2 + I_{ext} + g_e (E_e - V) + g_i (E_i - V) $$

# Izhikevich Neuron

The Izhikevich dynamics used in pinglab are:

$$ \frac{dV}{dt} = 0.04V^2 + 5V + 140 - U + I_{ext} + g_e (E_e - V) + g_i (E_i - V) $$
